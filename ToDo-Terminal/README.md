# ✅ ToDo-Terminal

A simple terminal-based To-Do list application built with Python.

---

## 📌 Features
- ➕ Add tasks  
- 📋 View all tasks  
- 🗑️ Delete tasks  
- ✏️ Edit existing tasks  
- 🏷️ Set task level (Low, Medium, High)
