# ✅ Four-Operation Calculator

A simple terminal-based calculator written in Python — supports the **four basic arithmetic operations** with an easy-to-use interface.

---

## 📌 Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📜 Display results for each operation
